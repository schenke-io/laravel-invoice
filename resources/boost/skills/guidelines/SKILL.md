---
name: guidelines-management
description: Manage and harmonize AI guidelines and skills within the package.
---

### When to use
- When you need to create or update the `core.blade.php` guidelines.
- When you need to create or update `SKILL.md` files for domain-specific features.
- When you need to ensure the documentation is consistent with the code.

### Features
- **Guideline Auditing**: Scan code and compare with existing guidelines to ensure ~3% density and accuracy.
- **Skill Generation**: Create new `SKILL.md` files for identified core domains.
- **Snippet Management**: Ensure code snippets in guidelines use `@verbatim` and `<code-snippet>` tags.
